# WeBlog小程序端

# 快速开始

1. 下载nodejs环境
2. 打开小程序项目文件夹uni-app-cli,执行`npm install`
```bash
cd uni-app-cli
npm install
npm run build
```
3. 打开微信开发者工具，导入项目文件夹`uni-app-cli/dist/build/mp-weixn`

## 小程序项目配置

如需修改关于小程序项目配置，建议直接修改`manifest.json`文件

## 请求根地址及密钥配置

开发环境修改`.env.development`,正式环境修改`.env.production`
